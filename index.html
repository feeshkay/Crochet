Paste or load a CraftBase export file. Yarn materials (Fiber/Yarn category) and crochet products will be merged — existing records with matching IDs are updated, new ones added.
How it works
1
In Crochet Studio: tap Export → copies JSON to clipboard or downloads file
2
In CraftBase: go to Import/Export → Crochet Sync → paste or upload the file
3
CraftBase merges: yarn → materials, projects → products, patterns saved
4
Reverse sync: export from CraftBase, import here to pull in material updates
